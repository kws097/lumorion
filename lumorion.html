Snabbstart Python!

Lär dig sätta upp en IDE för Python och skapa ett program som skriver ut Hello World!

  1. Installera Pythons senaste version Ladda ner
  2. Installera PyCharm Community Ladda ner
  3. Starta upp PyCharm och välj Python version 3.13 som din interpreter
  4. Skapa ett nytt projekt och döp det till "Hello World"
  5. Skriv i kodredigeraren: print("Hello World!") och tryck sedan på Run
  6. "Hello World" kommer då att skrivas ut i Konsolen
  7. Du har nu lärt dig hur man skapar ett Python-program som skriver ut text! Bra jobbat och lycka till med nästa projekt!
